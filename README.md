# 🌍 Cross-Border Loyalty Point Transfers

Welcome to the coolest way to transfer loyalty points across borders using blockchain wallets! This project lets users transfer, exchange, and redeem loyalty points across different programs on the Stacks blockchain.

## ✨ Features

🔄 Transfer points between users globally  
💱 Exchange points across loyalty programs  
🎟 Redeem points for rewards  
🔍 Track point balances and history  
🛡 Secure transactions with blockchain  
🌐 Interoperable with multiple programs  
🔐 Wallet-based authentication  

## 🛠 How It Works

**For Users**  
- Connect your Stacks blockchain wallet  
- Register with a loyalty program  
- Transfer points to another wallet  
- Exchange points between programs  
- Redeem points for rewards  

Boom! Your points are now globally accessible  

**For Verifiers**  
- Use get-balance to check user points  
- Call get-transaction to verify transaction history  

That's it! Instant verification
